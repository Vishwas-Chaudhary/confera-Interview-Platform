This is a 5th Semester **PBL project**, developed as a full-stack web application and **deployed on [Vercel](https://vercel.com)**.

## Getting Started

To set up and run this project locally, follow the steps below:

### Step 1 – Clone the repository
```bash
git clone https://github.com/Vishwas-Chaudhary/confera-Interview-Platform.git
```
### Step 2 - Navigate into the project folder
```bash
cd <project-folder>
```
### Step 3 - Install dependencies
```bash
npm install
```
### Step 4 – Run the servers
To run this project locally, open **two terminals** in your project folder.
### Terminal 1 – Run the frontend server
```bash
npm run dev
```
### Terminal 2 – Run the frontend server
```bash
npm run convex dev
```
