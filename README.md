 This is a 5th Semester **PBL project**, developed as a full-stack web application and **deployed on [Vercel](https://vercel.com)**.

## Getting Started

To run this project locally, open **two terminals** in your project folder.

### Terminal 1 – Run the frontend server
```bash
npm run dev
```
### Terminal 1 – Run the frontend server
```bash
npm run convex dev
```
